# Educational Backend Challenge (by Zahra Mustefa)

## 🧠 Overview
A small backend system that:
- Creates users
- Saves user progress
- Retrieves all progress
- Works offline (caches progress when offline and syncs later)

## 🧰 Stack
- Node.js
- Express
- SQLite3
- Offline caching with local JSON file

## ⚙️ Run Locally
```bash
npm install
node server.js
